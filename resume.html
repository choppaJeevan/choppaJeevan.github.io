OmniData: Agentic AutoML & Conversational Analytics Engine

Built an agentic AutoML system that takes any raw CSV and runs end to end with zero manual code changes: LLM-based dataset profiling, automated preprocessing with leakage guards, and a model tournament across 10+ algorithms. LLaVA inspects diagnostic plots and Llama 3 interprets metric deltas to generate the final model report; all LLM-generated code runs sandboxed with a self-correction loop. Mar 2026 - May 2026

DeepRAG: Retrieval-Augmented Generation (RAG) Pipeline

Built a RAG pipeline for 250+ page research PDFs using LlamaParse parsing and semantic chunking via Gemma-3. Cut vector storage and search latency ~3× with Matryoshka Representation Learning, and implemented hybrid search (dense + BM25) in a Dockerized Weaviate database with multi-query expansion and cross-encoder reranking. Evaluated with RAGAS on a held-out question set: 0.98 context recall and 0.93 context precision, with DeepSeek-R1 as the generation model.

May 2026

DataAdapt: Fine-Tuned Data Science Code Assistant

Fine-tuned Llama-3-8B into a domain-specific data science code assistant, building the full training-data pipeline to curate and ChatML-format a 50K-example instruction corpus. QLoRA with 4-bit quantization cut GPU memory use by ~75%, making training feasible on a single consumer GPU. Benchmarked against base Llama-3-8B on SQL and pandas tasks with ROUGE-L and BLEU, and mixed in regularization data to reduce catastrophic forgetting.
