Project Report

OmniData: Agentic Auto-ML

  • Architected a multi-agent Machine Learning orchestration system that autonomously ingests datasets, infers target variables via LLM semantic analysis, and executes a model tournament across 10+ algorithms.
  • Engineered a robust, leakage-immune preprocessing abstraction layer that fits imputers, RobustScalers, and encoders exclusively on training data, applying a two-phase correlation and multivariate leakage guard to eliminate data contamination.
  • Designed a dataset-agnostic configuration architecture using a Pydantic state schema and deterministic routing gateway, enabling the pipeline to process entirely distinct datasets with zero manual code modifications.
  • Integrated a Critic/Validator agent that computes train-vs-test score gaps, generates diagnostic plots, and explicitly flags overfitting, underfitting, or leakage conditions.
  • Orchestrated a multi-modal AI Judge system, utilizing LLaVA to visually detect non-linearity in OLS residual plots and Llama 3 to parse Macro F1 deltas.

Project Report

DeepRAG: Enterprise Analysis

  • Architected an enterprise-grade Retrieval-Augmented Generation (RAG) pipeline, processing complex 250+ page PDFs via LlamaParse to seamlessly preserve highly structured Markdown tables and layouts.
  • Replaced rigid token-window splitting with dynamic Semantic Chunking (via Gemma-3), accurately partitioning documents by contextual shifts to drastically enhance semantic boundaries.
  • Optimized vector storage overhead and search latency by 3x via Matryoshka Representation Learning (MRL).
  • Engineered a robust Weaviate v4 vector database supporting Hybrid Search, augmented by Multi-Query Expansion and advanced Cross-Encoder Reranking.
  • Orchestrated DeepSeek-R1 as the core reasoning engine.

DataAdapt: DS Copilot

  • Fine-tuned a Llama-3-8B base model into a specialized data science copilot capable of generating SQL queries, Pandas workflows, and statistical interpretations using Supervised Fine-Tuning (SFT).

  • Built a rigorous data-cleaning curation pipeline with Pandas to eliminate syntax noise across a 50k+ row training corpus.
  • Mitigated catastrophic forgetting by integrating a balanced regularization dataset.
