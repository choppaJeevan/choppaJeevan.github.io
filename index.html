OmniData: Agentic Auto-ML & Conversational Analytics Engine

  • Architected a multi-agent Machine Learning orchestration system that autonomously ingests datasets, infers target variables via LLM semantic analysis, and executes a model tournament across 10+ algorithms (XGBoost, LightGBM, Random Forest).
  • Engineered a robust, leakage-immune preprocessing abstraction layer that fits imputers, RobustScalers, and encoders exclusively on training data, applying a two-phase correlation and multivariate leakage guard to eliminate data contamination.
  • Designed a dataset-agnostic configuration architecture using a Pydantic state schema and deterministic routing gateway, enabling the pipeline to process entirely distinct datasets with zero manual code modifications.
  • Integrated a Critic/Validator agent that computes train-vs-test score gaps, generates diagnostic plots (ROC-AUC, Precision-Recall), and explicitly flags overfitting, underfitting, or leakage conditions.
  • Orchestrated a multi-modal AI Judge system, utilizing LLaVA to visually detect non-linearity in OLS residual plots and Llama 3 to parse Macro F1 deltas, replacing subjective human judgment with reproducible architectural selection.
  • Embedded a sandboxed Python execution engine with LLM-driven self-correction loops (up to 3 retries), enabling the system to autonomously generate, execute, and debug diagnostic scripts without human intervention.

DeepRAG: An Optimized RAG System for Research Document Analysis

  • Architected an enterprise-grade Retrieval-Augmented Generation (RAG) pipeline, processing complex 250+ page PDFs via LlamaParse to seamlessly preserve highly structured Markdown tables and layouts.
  • Replaced rigid token-window splitting with dynamic Semantic Chunking (via Gemma-3), accurately partitioning documents by contextual shifts to drastically enhance semantic boundaries and retrieval accuracy.
  • Optimized vector storage overhead and search latency by 3x via Matryoshka Representation Learning (MRL), efficiently truncating nomic-embed-text dense vectors to 256 dimensions.
  • Engineered a robust Weaviate v4 (Dockerized) vector database supporting Hybrid Search (Dense + BM25), augmented by Multi-Query Expansion and advanced Cross-Encoder Reranking.
  • Orchestrated DeepSeek-R1 (via Ollama) as the core reasoning engine, fusing reranked context chunks with high-parameter linguistic reasoning to produce heavily grounded, source-cited responses.
  • Validated end-to-end system architecture using an automated RAGAS evaluation suite, achieving near-perfect retrieval with 0.98 Context Recall and 0.93 Context Precision.

DataAdapt: Fine-Tuned Domain-Specific Data Science Copilot

  • Fine-tuned a Llama-3-8B base model into a specialized data science copilot capable of generating SQL queries, Pandas workflows, and statistical interpretations using Supervised Fine-Tuning (SFT) and instruction datasets formatted into strict ChatML templates.
  • Engineered a parameter-efficient training pipeline using QLoRA (Quantized Low-Rank Adaptation) via Hugging Face PEFT and bitsandbytes, reducing GPU memory footprint by 75% and enabling training on a single consumer GPU.
  • Built a rigorous data-cleaning curation pipeline with Pandas to eliminate syntax noise, enforce token length constraints, and filter out low-quality synthetic queries across a 50k+ row training corpus.
  • Mitigated catastrophic forgetting by integrating a balanced regularization dataset during the training loop, preserving the model's core reasoning and linguistic capabilities.
