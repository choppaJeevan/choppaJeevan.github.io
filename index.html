Jeevan Sai Reddy Choppa

MS in Data Science @ University of Maryland · LLM & Machine Learning Engineer

Hi, I'm Jeevan — I'm genuinely fascinated by how large language models work, and I've turned that fascination into a habit of building. I recently completed my MS in Data Science at the University of Maryland (GPA: 3.87/4.0), and these days I spend most of my time deep in the AI engineering stack: retrieval systems, fine-tuning, agents, and everything needed to make LLMs actually useful in production.

The best way I learn is by building end to end, and three projects capture that. DeepRAG is a Retrieval-Augmented Generation pipeline for 250+ page research PDFs that combines hybrid search, semantic chunking, and cross-encoder reranking to hit 0.98 context recall while cutting vector search latency 3×. With DataAdapt, I fine-tuned Llama-3-8B into a data science code assistant using QLoRA, curating my own 50K-example instruction corpus and slashing GPU memory needs by ~75%. And OmniData is an agentic AutoML engine that takes any raw CSV and autonomously profiles, preprocesses, and runs a model tournament across 10+ algorithms — validated on 5 datasets with zero code changes.

I care a lot about doing this work properly — leakage guards, sandboxed LLM-generated code, held-out evaluations, and metrics I can defend. I'm always learning something new in this space, and I'm looking for Data Science and AI/ML Engineering roles where I can keep growing while bringing that same end-to-end ownership to real production systems.
